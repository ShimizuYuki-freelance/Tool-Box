English description
This file is the code required to insert the OGP meta tag without using a plugin.
The part you want to fill in yourself
➀: Image URL when there is no thumbnail
➁: Twitter @username
➂: Content creator's Twitter @username
➃:Facebook App-ID
➄:Facebook adminID
is.
If you have any tags you would like to add, please add them accordingly.
